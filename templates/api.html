Text Analysis Endpoint

POST
/api/detect

Request

{
    "text": "The text you want to analyze..."
}

Response

{
    "isHuman": false,
    "confidence": 0.87,
    "perplexity": 25.42,
    "explanation": "The text perplexity score is 25.42...",
    "timestamp": "2023-07-15T14:30:00Z"
}

Example using curl

curl -X POST \
  https://yourdomain.com/api/detect \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -d '{"text":"The text you want to analyze..."}'
